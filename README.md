# Flanki
Jak uruchomić:
  Gra wymaga samodzielnego pobrania biblioteki graficznej SFML 3.0.2 oraz samodzielnej kompilacji.
 
  Biblioteke możesz pobrać tutaj: https://www.sfml-dev.org/download/
  
  Utwórz folder 'vslibs' na dysku C następnie rozpakuj w nim folder zip z biblioteką.
  Poprawna ścieżka do biblioteki powinna wyglądać w ten sopsób:
  C:\vslibs\SFML-3.0.2

  Aby zmienić miejsce biblioteki należy zmienić ścieżę prowadzącą do niej w IDE.
  Poradnik jak to zrobić znajduje się na tej stronie: https://www.sfml-dev.org/tutorials/3.0/getting-started/visual-studio/
  
  Nastepnie pobierz i rozpakuj Flany-main.zip.
  W folderze znajduje się plik flany.slnx otwórz go i skompiluj (zalecane visual studio 2026)
  
  Po skompilowaniu gra uruchomi się automatycznie.
  Miłej zabawy
